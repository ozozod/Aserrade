## Especificaciones Técnicas para el Logo

### Formatos Necesarios:
- **ICO**: Para el icono de Windows (256x256, 128x128, 64x64, 48x48, 32x32, 16x16)
- **PNG**: Para la aplicación (512x512, 256x256)
- **SVG**: Para escalado sin pérdida (opcional pero recomendado)

### Tamaños Mínimos:
- **Favicon**: 32x32 píxeles (mínimo)
- **Icono de app**: 256x256 píxeles (recomendado)
- **Logo completo**: Ancho mínimo 400px, altura variable

### Paletas de Color Sugeridas:

**Opción A - Profesional Azul:**
- Azul oscuro: `#2c3e50`
- Azul medio: `#3498db`
- Verde: `#27ae60`
- Blanco: `#ffffff`

**Opción B - Madera y Naturaleza:**
- Marrón madera: `#8b4513`
- Verde forestal: `#2ecc71`
- Beige claro: `#f5f5dc`
- Gris oscuro: `#34495e`

**Opción C - Moderno y Tech:**
- Azul marino: `#1e3a5f`

- Naranja: `#e67e22`

El logo debe incluir:

## Uso del Logo en la App

Una vez generado el logo, necesitarás:

1. **Crear múltiples tamaños**:
   - `icon.ico` (Windows): 256x256, 128x128, 64x64, 48x48, 32x32, 16x16
   - `icon.png` (app): 512x512 y 256x256
   - `logo.png` (documentos): ancho mínimo 400px

2. **Colocarlos en**:
   - `build/icon.ico` (para el instalador)
   - `build/icon.png` (para la app)
   - `public/logo.png` (para usar en la app si lo deseas)

3. **Convertir a ICO** (si solo tienes PNG):
   - Usar herramienta online: https://convertio.co/png-ico/
   - O: https://icoconvert.com/
