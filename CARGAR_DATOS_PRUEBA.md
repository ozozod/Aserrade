# 🧪 Cargar datos de prueba (MySQL)

## Opción recomendada

1. Abrí phpMyAdmin (o cliente MySQL)
2. Seleccioná tu base (`aserradero_db` o dev)
3. Importá un `.sql` de prueba desde `hostinger/` (u otro dump tuyo)

## Notas

- Revisá FKs/orden si el script asume tablas vacías.
- En producción, evitá datos de prueba.
